# Maryland Neighborhood Business Works Program Analysis

## Project Overview

This project analyzes Maryland's Neighborhood Business Works (NBW) Small Business Lending Program using public data from FY2011–FY2023.

The objective was to transform raw public data into executive-level insights through descriptive statistics and data visualization.

---

## Project Highlights

- 307 funded projects
- 67 Maryland cities
- $118.9M total funding
- 4,668 jobs created

---

## Tools Used

- Microsoft Excel
- Descriptive Statistics
- Data Visualization
- Executive Reporting

---

## Key Insights

- Baltimore accounted for the largest concentration of funded projects.
- Most grants were below $100K.
- Grant amounts were highly skewed by a small number of very large awards.
- Median grant size better represents the typical project than the mean.
- Geographic reach was broad but uneven across communities.
